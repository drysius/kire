# @kirejs/markdown

`@kirejs/markdown` renders Markdown files or strings inside Kire.

## Purpose

- convert markdown to HTML
- render markdown views with Kire locals
- expose helper directives for markdown workflows

## Main Helpers

- `kire.mdrender(content)`
- `kire.mdview(path)`
- `@markdown(source)`

## Typical Use

Use for docs sites, content pages and hybrid markdown + template rendering.
