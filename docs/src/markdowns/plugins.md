# Plugins

Kire has an extensible plugin system.

## Official Plugins

- **@kirejs/tailwind**: Compile Tailwind CSS on the fly.
- **@kirejs/assets**: Manage and optimize static assets.
- **@kirejs/ssg**: Static Site Generator.
- **@kirejs/markdown**: Render Markdown content.
- **@kirejs/iconify**: Use Iconify icons easily.
